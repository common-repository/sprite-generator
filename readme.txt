== Description ==



Create one sprite from different images to optimize your website loading! You can create JPG, PNG and GIF sprites.



== Installation ==



1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Go to the Sprite Generator menu and create your sprite

4. Go to your post, page or any other content and click 'Add a sprite image' above the content editor
